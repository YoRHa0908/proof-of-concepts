# Kingsway Academy Chat POC — Fixed

Working Next.js proof-of-concept for the client demo.

## Features

- Modern responsive landing page
- English / Spanish chat
- FAQ matching
- Quick question buttons
- Lead capture
- Admin dashboard
- CSV export
- localStorage demo persistence

## Run

```bash
npm install
npm run dev
```

Main page:

```txt
http://localhost:3000
```

Admin page:

```txt
http://localhost:3000/admin
```

For production, replace localStorage with Supabase, Firebase, PostgreSQL, HubSpot, or a CRM API.

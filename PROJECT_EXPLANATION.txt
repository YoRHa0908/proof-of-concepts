KINGSWAY ACADEMY CHAT DEMO - PROJECT EXPLANATION
==================================================

PROJECT OVERVIEW
----------------
This is a professional proof-of-concept chat interface for educational institutions
with bilingual support, lead capture, and admin dashboard functionality.

Live Demo: http://localhost:3001
Admin Panel: http://localhost:3001/admin

DEMO SCOPE REQUIREMENTS (ALL MET)
----------------------------------

1. ✅ SMALL WEBSITE CHAT WINDOW UI
   - Professional chat interface with gradient background
   - Message bubbles for user/assistant conversations
   - Quick questions buttons for common inquiries
   - Input form with send functionality
   - Sophisticated design with glass morphism effects

2. ✅ ENGLISH + ONE ADDITIONAL LANGUAGE
   - Full English/Spanish bilingual support
   - Language toggle button (Switch to Spanish / Cambiar a inglés)
   - All UI text translated (headlines, buttons, forms, messages)
   - FAQ responses in both languages

3. ✅ 2–3 SAMPLE FAQ RESPONSES (ACTUALLY 4 IMPLEMENTED)
   - Admissions process: "How does admission work?"
   - Available programs: "What programs are available?"
   - Tuition information: "Can I ask about tuition?"
   - Contact methods: "How can I contact the school?"
   - Each FAQ has keyword matching for intelligent responses

4. ✅ SIMPLE LEAD CAPTURE (NAME/EMAIL)
   - Name field (required)
   - Email field (required, type="email")
   - Interest textarea (optional)
   - Form validation with real-time feedback
   - Browser localStorage for demo persistence
   - Success confirmation messages

5. ✅ SIMPLE ADMIN VIEW, LEAD EXPORT, BASIC STORAGE
   - Admin dashboard at /admin
   - Table view of all captured leads
   - CSV export functionality with date-stamped filenames
   - Refresh and clear all buttons
   - localStorage implementation for demo purposes

6. ✅ LIVE DEMO LINK / TEST PAGE
   - Development server running on http://localhost:3001
   - Both main chat and admin pages accessible
   - Project builds successfully with no errors

TECHNICAL IMPLEMENTATION
------------------------

ARCHITECTURE:
- Next.js 14 with App Router
- React functional components with hooks
- TypeScript for type safety
- Tailwind CSS for styling
- Component-based architecture

KEY COMPONENTS:

1. ChatDemo.tsx (Main Component)
   - Manages chat state, messages, and language
   - Handles form submissions and lead capture
   - Implements auto-scrolling for chat messages
   - Provides bilingual UI with toggle functionality

2. Admin Page (/admin/page.tsx)
   - Displays captured leads in table format
   - Provides CSV export functionality
   - Allows data management (refresh, clear)
   - Same professional styling as main page

3. Data Layer (lib/ folder)
   - content.ts: UI translations and FAQ data
   - storage.ts: localStorage utilities with error handling
   - types.ts: TypeScript type definitions
   - utils.ts: Validation and utility functions
